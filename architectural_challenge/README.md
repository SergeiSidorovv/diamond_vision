1. Домены и репутация

Основной домен + поддомены по клиентам/направлениям: client1.domain.ru, sales.domain.ru.
Для каждого поддомена: SPF, DKIM, DMARC (p=none на старте), корректный unsubscribe link.

2. Поставщики доставки

Основной: российский SMTP-провайдер (например, UniSender, Mailigen, Sendpulse).
Резервный: другой российский провайдер для автоматического переключения при проблемах.
При крупных потоках — выделенные IP, постепенный warm-up.

3. Приложение

Развёртывается на 1–2 VPS Selectel.
Компоненты: backend (Python/Node), база PostgreSQL, очередь (Redis/RabbitMQ), воркеры для отправки писем.
Логика: выбор провайдера → рендер шаблона → отправка → логирование статусов → обработка bounce/complaints → suppression list.

4. Отказоустойчивость и мониторинг

Автоматическое переключение на резервного провайдера при падении основного.
Rate-limiting по IP и поддомену, seed-листы для проверки deliverability.
Метрики и алерты: открытие, клики, bounces, жалобы.

5. Операции и безопасность

Разделение потоков для скорости ответов.
Обработка отписок, жалоб, spam-complaints.
Хранение согласий пользователей.

6. Пример стоимости (ориентировочно)

VPS Selectel: 2×200–400 ₽/мес.
SMTP-провайдеры: 1–2 тыс писем/день ≈ 1–3 тыс ₽/мес.
Выделенные IP при необходимости — 1–2 тыс ₽/мес.
Итого: ~4–7 тыс ₽/мес при умеренных объёмах, с высокой отказоустойчивостью.
